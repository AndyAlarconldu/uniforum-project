# 📊 Analysis Domain - UniForum Project

The `analysis-domain` is responsible for collecting, processing, and visualizing data generated by user activity across the UniForum platform. It enables the platform to provide valuable insights such as usage statistics, popular discussion topics, and interaction trends.

---

## 📁 Structure
```
analysis-domain/
│
├── heatmap-service/ # Generates activity heatmaps by time/day/user
├── metrics-service/ # Aggregates global platform metrics and statistics
└── top-topic-service/ # Identifies and ranks the most discussed topics
```

---

## 🧩 Services Overview

### 🔥 `heatmap-service`
- Visualizes user engagement over time (e.g., posts per hour, day, or course).
- Useful for detecting peak activity hours and inactive periods.

### 📈 `metrics-service`
- Provides overall platform usage stats.
- Tracks number of users, posts, replies, votes, and login frequency.
- Supports periodic report generation.

### 🏆 `top-topic-service`
- Ranks most discussed topics based on replies, votes, and views.
- Can filter by date range or category.
- Helps users discover trending content.

---

## 🛠 Technologies Used

- **Python + FastAPI**
- **PostgreSQL / MongoDB**
- **Pandas / NumPy (for analysis)**
- **RabbitMQ** (event-driven updates)
- **Docker** (containerized services)
